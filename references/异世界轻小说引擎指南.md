# 异世界轻小说引擎指南

本文档为 isekai.md 的完整内容，是 SKILL.md Part C 的详细参考。

> **定位**：日本商业轻小说常用的两套写法引擎——战记/征服引擎 + 学园/异能引擎。写自己的作品时：先选引擎，再混模块。

---

## 总纲

### 共通底盘
短段推进、对话占比高、焦点锁定、内心可见、地位语体分层、先给结果再补道理、章/卷有钩子。

读者路径：听见人说话 → 看见场面 → 钻进某人脑袋 → 被设定/误判打一拳 → 留下下一口悬念。

### 写作优先顺序
1. 选引擎（战记征服 / 学园异能 / 混合） 2. 锁视角高度 3. 定本章情绪主菜 4. 选句癖模块（勿全开）

## 引擎对照

| 维度 | 战记/征服 | 学园/异能 |
|------|-----------|-----------|
| 爽 | 降维、压迫、制度碾压 | 概念对撞、价值观击穿 |
| 笑 | 表面威严vs内里慌乱；忠诚过度解读 | 不幸体质、废柴经济、角色崩坏跑题 |
| 暖 | 小人物被保护、忠诚日常 | 憧憬与守护、普通人的手 |
| 视角 | 第三人称有限，场景切焦点 | 同左，主角吐槽密度更高 |
| 内心 | (……) 或叙述嵌入 | (……) + 长段自问自答 |
| 信息差 | 同场双真相（忠诚滤镜） | 多方各知一半，读者居中 |
| 设定流出 | 口述史、战斗需要、误判 | 课堂式拆招、专有名词后括英文 |
| 多线 | 阵营镜头切换 | 序章/章/行间/终章制度性切线 |

## 行文风格

### 章节结构

**学园异能型**：序章(异常日常)→第1-4章(主线升级)→行间(他视角短切)→终章(情绪结算+小奇迹)

**战记征服型**：冷开场对话→轻分节→时间回溯或换阵营→场景内动作短句→对话→括号心理→章末未决意图

### 段落句式
- 1-4句成段为主；动作短句；情报中句；设定补丁立刻回到人物反应
- 战记偏好：技能`［］`、大招`『』`、数字锚+体感锚
- 学园偏好：解释连接词(也就是说/说白了)、递进吐槽、比喻链、感叹拖长、专名双标

### 场景建立
- 开场：战记用冲突台词/劳动汗水；学园用时间地点+生活抱怨
- 描写原则：功能描写，非风景文。新势力立刻进冲突，人物1视觉锚+1地位标签+1习惯

### 战斗行文
四层交错：动作(高)+技能/概念(改变场面)+策略(当场判断)+情感/信念(收束)
不靠招式说明书，而靠「预估vs现实」落差。无战斗章可用劳动/交易维持紧张。

## 对话风格

### 语体分层
| 类型 | 特征 |
|------|------|
| 至高统治者表面 | 短、稳、少解释 |
| 狂热部下 | 敬语拉满+过度解读 |
| 废柴主角 | 大呼小叫、自我吐槽、突然认真 |
| 傲娇同伴 | 毒舌、害羞暴力、义理 |
| 天然吐槽役 | 短句、直觉正确 |
| 理解者 | 冷评、担保、神话口吻 |
| 村姑/底层 | 结巴感谢、自我贬低 |

### 对话句型
开场钩子/地位压制/信息差双轨/忠诚信仰/底层感谢/战斗喊招/说明+事故/降维翻译/夸张内心/情报钉死/价值观锤句/终章生活回响/喜剧插入

### 反对话墙
A说话→B说话→A说话→——插入动作/气场/环境/第三人/心声→B说话→A锤句

## 模板

### A 异世界战记/远征
冷开场→锚焦点→双线→展示性事件→认知翻面→收束钩子

### B 公会/忠诚日常
据点地理→琐事建性格→支配忠诚互动→聚会桌闲聊→口述圣史→不安决心

### C 底层成长
身体劳动→社群现状→小善互惠→外出遭遇→名号外界生长→上层短切

### D 学园/异能一卷戏
时间戳抱怨→轻喜剧建关系→微型说明→事故化→危机升级→行间→决战拆招→终章生活回响

### E 混合（推荐）
骨架用学园型，压迫制度用战记型，每卷只选1个主题锤句。

## 自检清单

### 行文
- 选了引擎？开场是对话/身体感/冲击而非空景？焦点唯一清晰？
- 设定经口述/误判/战斗流出？有表里裂缝？战斗四层交错？
- 学园型有行间短切？终章回收生活感/信念？

### 对话
- 遮名辨人？连续对白≤4轮打断？有地位差？
- 至少一次打断/抢话？技能名只在改变场面时出现？
- 底层角色避免无来源系统黑话？至少一句可记锤句？

### 主题(每卷1-2个)
憧憬/守护/忠诚/征服/日常尊严——写进角色行动而非旁白。
