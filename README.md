## 我们的核心理念

长篇小说创作中，AI 面临两个根本矛盾：

**矛盾一：AI 记不住** — 上下文窗口有限，写到后面忘了前面的人设、伏笔、战力设定。

**矛盾二：AI 写不长** — 字数不够时 AI 倾向于"原地堆砌描写"而非"往前推进剧情"，导致篇幅缩水但故事毫无进展。

这个技能就是为了解决这两个矛盾而设计的。

## 如何让 AI 驾驭长文本而不遗忘

### 全量标准化 Markdown 持久化架构

我们不依赖 AI 的上下文记忆，而是将所有关键信息**持久化到磁盘文件**中，每卷创作时重新加载。

```
project/
├── global_characters.md          # 全局角色静态属性库
├── global_timeline.md            # 全局时间轴总纲
├── global_foreshadowing.md       # 全局伏笔追踪总表
├── global_story_outline.md       # 全局故事主线总纲
├── Volume_01_卷名/
│   ├── Volume_01_text.md         # 本卷正文
│   ├── Volume_01_characters.md   # 本卷角色卡
│   └── Volume_01_summary.md      # 本卷梗概+字数归档
└── ...
```

### 三大防遗忘机制

**① 就近完整继承规则**
创作新一卷时，每个角色的角色卡必须从最近一卷完整复制全部过往经历，再追加本卷新信息。AI 无需记忆——每次打开角色卡就是角色的完整一生。

**② 全局文件 → 分卷文件 双重校验**
`global_characters.md` 保存恒定设定（种族、本源能力等），永久固定。各卷角色卡保留动态状态，逐卷可追溯。人设判定优先级：最新分卷角色卡 > 全局角色静态库。

**③ 伏笔追踪总表**
所有伏笔统一登记在 `global_foreshadowing.md`。创作前先读取未回收伏笔清单，确保每条伏笔都有归宿。

## 字数不够时的扩写规则

AI 字数不足时，绝大多数问题出在一个根因：**AI 倾向于把已有场景描写得更细致，而不是增加新的故事事件。**

我们的规则体系围绕"强制故事增量、禁止描写堆砌"展开。

### 第一层：前置预防——容量预估

创作正文前先输出容量预估表，将字数目标转化为故事情节数量：

```markdown
【本卷容量预估表】
- 目标总字数：80000 字 | 计划话数：序章 + 12 话正篇 + 尾声
- 所需最小剧情节拍数：36 个（= 12 话 × 3）
- 已规划剧情节拍数：28 个 → 缺口 8 个 → 需扩容！
```

### 第二层：字数感知锚点

每话大纲嵌入 5 项锚点：目标字数 / 核心剧情节点数 / 对话场景数 / 描写段落数 / 情绪转折点数。未全部覆盖 = 字数大概率不达标。

### 第三层：强制续写 + 有效增量检测

每话完成后立即统计字数。若未达标：

```
【字数核验】本话当前 3210 字，目标下限 3500 字，缺口 290 字，启动强制续写。
```

**铁律一**：只能新增故事事件，禁止描写堆砌。
**铁律二**：有效增量 < 70% 则续写无效，重做。

### 第四层：扩写方向智能分流（核心机制）

**不再一刀切。先诊断内容完成度，再决定扩写方向。**

| 完成度 | 判断标准 | 扩写方向 |
|--------|---------|---------|
| **严重不足** (<40%) | 骨架式叙述，节点仅100-200字 | **内部丰富**：选2-3核心节点展开为完整场景 |
| **轻度不足** (40-70%) | 部分已展开但单薄 | **内部为主+小幅外延**：补层次，再插小型过渡 |
| **接近达标** (≥70%) | 各节点已展开 | **外部延伸**：节点间或末尾新增小事件 |
| **已达标** | 已达标但需扩容 | **仅外部延伸**：新增节点或插话 |

**示例**：计划3500字的一话，AI初稿仅800字（23%）。五个节点全是骨架。错误做法是在末尾加新事件凑字数。正确做法是先选"酒馆接任务"和"遭遇魔物"两个核心节点展开为完整场景，从800字扩至2800字，再用小型过渡事件补齐剩余缺口。

### 第五层：串行创作 + 下一话锁定

一次只写一话，**本话未达标不得启动下一话**。达标后锁定正文，后续不得回头修改。

---

## 量化指标

| 规则 | 阈值 | 后果 |
|------|------|------|
| 修饰性描写占新增字数 | > **30%** | 判定无效扩写 |
| 有效增量占新增字数 | < **70%** | 续写无效，重做 |
| 单话字数 | < **3500 字** | 启动强制续写 |
| 有效剧情字数占总字数 | < **70%** | 校验不通过 |
