## 五规范体系

| 规范 | 核心内容 |
|------|---------|
| **A 持久化** | 文件架构/字数管控/串行创作/有效增量 |
| **B 防注水** | 10种注水手段+删去测试/信息增量测试/节奏测试 |
| **C 写作风格** | 叙事视角/语言句式/对话节奏/情绪战斗 |
| **D 日轻排版** | 标点二选一/段落铁则/拟声词/内心独白20-30% |
| **E 异世界引擎** | 战记征服/学园异能/语体分层/5种模板 |

## 安装
