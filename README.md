## 两大规范

### 持久化规范（解决AI遗忘）
全局文件+分卷文件持久化架构，就近完整继承角色卡，伏笔追踪总表。

### 写作风格规范（保证轻小说品质）
叙事视角、语言句式、对话占比、节奏模板、情绪轮换、战斗四层——全部可执行。

## 安装
```bash
git clone https://github.com/LennyFace24/light-novel-creator-skill.git ~/.claude/skills/novel-persistence-skill
```

## 使用
```
/novel-persistence-skill init | preload | write | outline | archive | validate | wordcount-rescue
```
