## 三规范体系

| 规范 | 来源 | 核心内容 |
|------|------|---------|
| **持久化规范** | rule.md | 文件架构/字数管控/串行创作/有效增量 |
| **写作风格规范** | write.md | 叙事视角/语言句式/对话节奏/情绪战斗 |
| **异世界引擎指南** | isekai.md | 战记征服/学园异能引擎/语体分层/模板 |

先选引擎(战记/学园/混合)，再锁视角，再执行写作与字数规范。
